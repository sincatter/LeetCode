## Same Tree
Given two binary trees, write a function to check if they are the same or not.  
Two binary trees are considered the same if they are structurally identical and the nodes have the same value.  

### Example 1:

__Input__:  
```
    1         1
   / \       / \
  2   3     2   3

 [1,2,3],   [1,2,3]
```
__Output__: true
### Example 2:

__Input__:
```
     1         1
    /           \
   2             2

 [1,2],     [1,null,2]
```
__Output__: false
### Example 3:

__Input__: 
```
   1         1
  / \       / \
 2   1     1   2

[1,2,1],   [1,1,2]
```
__Output__: false

## 相同的树
给定两个二叉树，编写一个函数来检验它们是否相同。  
如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

### 示例 1:

__输入__: 
```
    1         1
   / \       / \
  2   3     2   3

 [1,2,3],   [1,2,3]
```
__输出__: true
### 示例 2:

__输入__:  
```
     1         1
    /           \
   2             2

 [1,2],     [1,null,2]
```
__输出__: false
### 示例 3:

__输入__:  
```
   1         1
  / \       / \
 2   1     1   2

[1,2,1],   [1,1,2]
```

__输出__: false
